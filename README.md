# CRM Aparcament Rector

Aplicacio estatica per gestionar els clients interessats de l'Aparcament Rector.

## Que permet fer

- Importar un Excel de clients interessats.
- Classificar clients per estat comercial.
- Filtrar per tipus d'interes.
- Assignar manualment una placa o traster a cada client.
- Guardar les dades al navegador.
- Exportar els clients a Excel.

## Com s'obre

Obre `index.html` en un navegador modern.
